# Projeto-ANGULAR-Faculdadee
# Copa 2026 ⚽

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

O objetivo do projeto foi transferir uma página desenvolvida inicialmente com **HTML e CSS** para o framework **Angular**.

Durante a migração, foram utilizados **componentes Angular** para organizar melhor a aplicação, como a criação de páginas separadas e um componente reutilizável de navegação (Navbar).

Também foi implementado o **Angular Router** para realizar a navegação entre as páginas do projeto.

## Tecnologias utilizadas

- Angular
- TypeScript
- HTML
- CSS
